HTML

¿que es y cual es su funcion?

HTML (HyperText Markup Language) es un lenguaje de marcado de hipertexto que se utiliza para estructurar y dar forma a las páginas web y su contenido. Se trata de un código que permite organizar textos, imágenes, videos, botones y otros elementos de navegación. HTML es la base de la World Wide Web y se utiliza en el 94% de los sitios web. Es un conocimiento indispensable para quienes se dedican a la creación de páginas web, ya sea de manera profesional o como hobby.

2. CSS



" CSS

¿que es y cual es su funcion?

CSS (Cascading Style Sheets) es un lenguaje de programación que se utiliza para dar formato a los elementos de un sitio web: Define los colores, fuentes, tamaños, posicionamiento y otros valores estéticos Se encarga de presentar visualmente los datos de un sitio web a través de un navegador Se usa para estilizar elementos escritos en un lenguaje de marcado como HTML El CSS complementa la información que forma parte de un sitio web. Mientras que el código en HTML incluye todos los datos, el código en CSS se encarga de darles formato.

¿que es y cual es su funcion?

JavaScriptes un lenguaje de programación que se utiliza para hacer que las páginas web sean interactivas y dinámicas: Permite agregar características como juegos, animaciones, efectos de estilo, y eventos que ocurren al presionar botones o ingresar datos en formularios Se ejecuta directamente en el navegador del usuario, por lo que no es necesario recargar la página para realizar tareas como actualizar contenido en tiempo real o validar formularios Es compatible con todos los navegadores modernos JavaScript es un lenguaje de alto nivel, orientado a objetos, interpretado y multiparadigma.
